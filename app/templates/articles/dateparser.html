Reminder Bot Dateparser





The Reminder Bot Dateparser, developed under BSD-3-Clause by scrapinghub, is now available no-install from our unlimited API. This is the same algorithm used by the $natural command.
The API may become limited in the future.

URL: https://dateparser.reminder-bot.com
Params (as JSON):

  • content Required Specify the string to be converted.
  • timezone Take input as specified timezone
  • to_timezone Convert output to specified timezone
This matches up to the docs available here.

Example (Python 3):

import requests

response = requests.post('https://dateparser.reminder-bot.com/', json={'content': 'The 5th of December 2019', 'timezone': 'Europe/Paris'})

assert response.json() == {'result': 1575500400.0} # Time returned as a UNIX timestamp (seconds)
