Files with Python

Introduction

Files, or some sort of 'persistent', non-volatile storage are key to the wide majority of applications. Python provides many ways to store different types of data in files that remain on the system after the program shuts down.

Opening a File

Before using a file, it must be opened in an explicit mode. Basically, you have to tell Python how to open the file- it can't decide for you. There's 3 modes you can open a file in:

  • Read
  • Write
  • Append

Read Mode

Read mode allows you to take data from the file without changing it.

Write Mode

Write mode allows you to completely remake a file from scratch.

Append Mode

Append mode allows you to add data to the end of a file.
